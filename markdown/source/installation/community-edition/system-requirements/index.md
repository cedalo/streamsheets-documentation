# System requirements

## CPUs

-   Minimum: 1 CPU
-   Recommended: 4 CPUs or more

## RAM

-   Minimum: 1 GByte RAM
-   Recommended: 8 GByte RAM or more

## Raspberry Pi

-   Minimum: Raspberry Pi 3b
-   Recommended: Rasberry Pi 3b+ or higher
