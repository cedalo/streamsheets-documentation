# OCT2DEC

Converts an octal number to decimal.

**Syntax**

=OCT2DEC(Number)

**Arguments**

| Name   | Description                  |
|--------|------------------------------|
| Number | The octal number to convert. |

**Return Value**

The decimal number or a \#NUM! error if given number is not a valid
octal number.

**Example**

| Function     | Result | Comment                      |
|--------------|--------|------------------------------|
| =OCT2DEC(54) | 44     | converts octal 54 to decimal |
