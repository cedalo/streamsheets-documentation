# UNICODE

Returns the unicode number for first character of given string.

**Syntax**

=UNICODE(Text)

**Arguments**

| Name | Description                                                      |
|------|------------------------------------------------------------------|
| Text | A string for which to return the unicode of its first character. |

**Return Value**

The unicode number of first character of specified text.

**Example**

| Function      | Result |
|---------------|--------|
| =UNICODE("A") | 65     |
