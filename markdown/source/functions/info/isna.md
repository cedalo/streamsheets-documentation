# ISNA

Checks, if given value represents an \#NA `error <error>`.

**Syntax**

=ISNA(Value)

**Arguments**

| Name  | Description     |
|-------|-----------------|
| Value | Value to check. |

**Return Value**

TRUE if given value represents an \#NA error, otherwise FALSE.

**Example**

| Function | Result | Comment           |
|----------|--------|-------------------|
| =ISNA(3) | FALSE  | No \#NA detected. |
