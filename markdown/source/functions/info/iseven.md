# ISEVEN

Checks, if given value is an even number.

**Syntax**

=ISEVEN(Value)

**Arguments**

| Name  | Description     |
|-------|-----------------|
| Value | Value to check. |

**Return Value**

TRUE if value is an even number, otherwise FALSE.

**Example**

| Function   | Result | Comment                 |
|------------|--------|-------------------------|
| =ISEVEN(3) | FALSE  | Three is no even value. |
