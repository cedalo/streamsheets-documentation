# ISODD

Checks, if given value is an odd number.

**Syntax**

=ISODD(Value)

**Arguments**

| Name  | Description     |
|-------|-----------------|
| Value | Value to check. |

**Return Value**

TRUE if value is an odd number, otherwise FALSE.

**Example**

| Function  | Result | Comment                |
|-----------|--------|------------------------|
| =ISODD(3) | TRUE   | Three is a odd number. |
