# MIN

Evaluates the minimum value of all given values.

**Syntax**

=MIN(Value1, Value2...)

**Arguments**

| Name              | Description                             |
|-------------------|-----------------------------------------|
| Value1, Value2... | Number values to evaluate minimum from. |

**Return Value**

Minimum value from all given values.

**Example**

| Function       | Result |
|----------------|--------|
| =MIN(3, 6, 12) | 3      |
