# MAX

Evaluates the maximum value of all given values.

**Syntax**

=MAX(Value1, Value2...)

**Arguments**

| Name              | Description                         |
|-------------------|-------------------------------------|
| Value1, Value2... | Number values to evaluate max from. |

**Return Value**

Maximum value from all given values.

**Example**

| Function       | Result |
|----------------|--------|
| =MAX(3, 6, 12) | 12     |
