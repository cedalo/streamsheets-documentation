# POWER

Evaluates the value of base to the power of exponent.

**Syntax**

=POWER(Value, Exponent)

**Arguments**

| Name     | Description            |
|----------|------------------------|
| Value    | Value to use for base. |
| Exponent | Exponent to use        |

**Return Value**

Value of base to the power of exponent.

**Example**

| Function     | Result |
|--------------|--------|
| =POWER(2, 3) | 8      |
