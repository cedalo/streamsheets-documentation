# ARCCOS

Calculates the arccosine value of the given value

**Syntax**

=ARCCOS(Value)

**Arguments**

| Name  | Description                      |
|-------|----------------------------------|
| Value | Value to evaluate arccosine for. |

**Return Value**

Evaluated angle in radians.

**Example**

| Function     | Result     |
|--------------|------------|
| =ARCCOS(0.3) | 1.26610367 |
