# ROUND

Rounds the value to the given amount of decimals.

**Syntax**

=ROUND(Value, Decimals)

**Arguments**

| Name      | Description                     |
|-----------|---------------------------------|
| Value     | Value to round.                 |
| Decimals. | Allowed decimals in the result. |

**Return Value**

Rounded value with the allowed number of decimals.

**Example**

| Function             | Result |
|----------------------|--------|
| =ROUND(7.3423423, 3) | 7.342  |
