# INT

Rounds a number value down to its smaller integer.

**Syntax**

=INT(Value)

**Arguments**

| Name  | Description                               |
|-------|-------------------------------------------|
| Value | Number value to round down to an integer. |

**Return Value**

Smaller integer of specified number value.

**Example**

| Function  | Result |
|-----------|--------|
| =INT(3.6) | 3      |
