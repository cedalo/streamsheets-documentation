# ARCSIN

Calculates the arcsine value of the given value

**Syntax**

=ARCSIN(Value)

**Arguments**

| Name  | Description                    |
|-------|--------------------------------|
| Value | Value to evaluate arcsine for. |

**Return Value**

Evaluated angle in radians.

**Example**

| Function     | Result     |
|--------------|------------|
| =ARCSIN(0.3) | 0.30469265 |
