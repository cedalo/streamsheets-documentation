# TRUNC

Truncates a number to an integer.

**Syntax**

=TRUNC(Value)

**Arguments**

| Name  | Description               |
|-------|---------------------------|
| Value | Number value to truncate. |

**Return Value**

Number value to truncate.

**Example**

| Function          | Result |
|-------------------|--------|
| =TRUNC(7.3423423) | 7      |
