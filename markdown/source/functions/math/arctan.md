# ARCTAN

Calculates the arctangent value of the given value

**Syntax**

=ARCTAN(Value)

**Arguments**

| Name  | Description                       |
|-------|-----------------------------------|
| Value | Value to evaluate arctangent for. |

**Return Value**

Evaluated angle in radians.

**Example**

| Function     | Result    |
|--------------|-----------|
| =ARCTAN(0.3) | 0.2914568 |
