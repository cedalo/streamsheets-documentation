# FRAC

Returns the fractional part of a number value.

**Syntax**

=FRAC(Value)

**Arguments**

| Name  | Description                                 |
|-------|---------------------------------------------|
| Value | Number value to get the fractional part of. |

**Return Value**

The fractional part of specified number value.

**Example**

| Function    | Result |
|-------------|--------|
| =FRAC(1.2)  | 0.2    |
| =FRAC(-1.2) | -0.2   |
| =FRAC(23)   | 0      |
