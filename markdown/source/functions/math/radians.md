# RADIANS

Converts the degrees value to radians.

**Syntax**

=RADIANS(Degrees)

**Arguments**

| Name    | Description               |
|---------|---------------------------|
| Degrees | Degrees value to convert. |

**Return Value**

Angle in radians.

**Example**

| Function      | Result  |
|---------------|---------|
| =RADIANS(180) | 3.14159 |
