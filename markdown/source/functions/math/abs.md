# ABS

Returns the absolute value of a number.

**Syntax**

=ABS(Value)

**Arguments**

| Name  | Description                                   |
|-------|-----------------------------------------------|
| Value | Value of which to return the absolute number. |

**Return Value**

The absolute number.

**Example**

| Function | Result | Comment                  |
|----------|--------|--------------------------|
| =ABS(-3) | 3      | Absolute number is three |
