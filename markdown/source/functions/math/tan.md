# TAN

Returns the tangent value of the angle.

**Syntax**

=TAN(Angle)

**Arguments**

| Name  | Description      |
|-------|------------------|
| Angle | Angle in radians |

**Return Value**

Evaluated tangent value.

**Example**

| Function     | Result |
|--------------|--------|
| =TAN(PI()/4) | 1      |
