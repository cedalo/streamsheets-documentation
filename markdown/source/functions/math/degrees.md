# DEGREES

Converts radian value to degrees.

**Syntax**

=DEGREES(Radians)

**Arguments**

| Name    | Description              |
|---------|--------------------------|
| Radians | Radian value to convert. |

**Return Value**

Degrees converted from radian angle.

**Example**

| Function       | Result |
|----------------|--------|
| =DEGREES(PI()) | 180    |
