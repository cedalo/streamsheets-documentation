# SIGN

Returns the sign of given number.

**Syntax**

=SIGN(Value)

**Arguments**

| Name  | Description     |
|-------|-----------------|
| Value | Value to check. |

**Return Value**

Returns 1, if number value is positive, -1, if it is negative or 0, if
value is 0.

**Example**

| Function  | Result |
|-----------|--------|
| =SIGN(-3) | -1     |
