Custom themes

Custom Themes allow you to modify the visual appearance of the Management Center. The below described settings alter the colors used to render the user interface and to define a title or logo to use for the login screen and the application itself.

The Theme has to be defined within the "config.json" of the Management Center. As there are two modes available for the visual appearance, "Dark" and "Light", there are also two sections available for modification. Following the content of a completely defined theme:

Login Screen​

Within the login screen, the definitions are used like shown in the image and explained below:

  • logo used at the top right of the login dialog. If not available, no logo will be displayed.
  • title used for the title. If left out 'Management Center' is used.
  • backgroundLogin Background Color for the screen.

Application​

Within the application, the definitions are used like shown in the image, displaying a fully customized app:

The entries in the config have the following effects:

  • title is the text of the title to display in the title bar, if 'titleBar' is not set to 'logo'.
  • titleBar If this value is set to 'logo' the logo will be used instead of the title for the application title
  • logo to use instead of title, if 'titleBar' is set to 'logo'
  • background Background Color for all pages.
  • backgroundTitleBar Background Color for title bar.
  • backgroundNavigation Background Color for the navigation bar
  • text Text color for all primary texts. These include most of the texts, except hints for edit boxes, breadcrumbs and other
  • textSecondary Text color for hints to edit boxes and breadcrumbs.
  • primary Color for user interface elements like check boxes, badges, focussed navigation items or buttons.
