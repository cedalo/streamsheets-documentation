ARCTAN2

Calculates the arc tangent value based on the given x and y coordinates.

Syntax

=ARCTAN2(X, Y)

Arguments

NameDescription
XX Coordinate.
YY Coordinate.

Return

Evaluated angle in radians.

Examples

FormulaResultComment
${formula}
0.78539816An equal value for X and Y will lead to a 45 degree angle.
