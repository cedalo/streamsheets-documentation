SUM

Evaluates the sum of all given values.

Syntax

=SUM(Range1, ...RangeN)

Arguments

NameDescription
Range1...RangeNA list of references or ranges to calculate the sum for.

Return Value

Sum of all given arguments.

Example

FunctionResult
=SUM(3, 6, 12)21
=SUM("Test3",A2:B4, 2)
