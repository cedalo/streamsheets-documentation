Simple Shapes

Simple shapes are shapes that do not handle user input to allow interactivity. Below a list of the shapes. The shapes in the bottom row are not simple shapes, but interactive shapes.

The shapes are created by selecting a shape with the mouse and placing the item on a sheet or dashboard. Some of the shapes provide a shape specific reshape marker. This differ from size or rotate markers in that they provide a shape specific resizing or shape relayout logic.

In the sample below the green markers represent those special markers. Here the markers influence the width of the arrow base and the length of the arrow head.
