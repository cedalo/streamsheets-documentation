# SUM

Evaluates the sum of all given values.

**Syntax**

=SUM(Range1, ...RangeN)

**Arguments**

| Name            | Description                                              |
|-----------------|----------------------------------------------------------|
| Range1...RangeN | A list of references or ranges to calculate the sum for. |

**Return Value**

Sum of all given arguments.
